# Advent of Code 2021

## Description

Advent of Code 2021 solutions written in Typescript using the [Nest](https://github.com/nestjs/nest) framework.  Most of these solutions are overly complicated.  I used this opportunity to explore various patterns and solutions.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
